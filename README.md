📈 C# Stock Portfolio Manager

Basit bir C# Console uygulaması.
Birden fazla kullanıcı için hisse portföyü yönetimi yapılabilir.

🚀 Özellikler

👤 Birden fazla kullanıcı oluşturma

📊 Kullanıcı hisselerini listeleme

➕ Hisse ekleme

❌ Hisse silme

🔁 Kullanıcılar arasında hisse transferi

💰 Kullanıcı net portföy değeri hesaplama

💾 Verileri dosyada saklama

🧠 Kullanılan Teknolojiler

C#

.NET Console Application

File I/O (Dosya okuma/yazma)

Object Oriented Programming (OOP)

📂 Veri Yapısı

Veriler veriler.txt dosyasında saklanır.

Örnek veri formatı:

Ali|THYAO,10,200;ASELS,5,80
Mehmet|GARAN,20,45

Format:

KullanıcıAdı|Sembol,Adet,Fiyat;Sembol,Adet,Fiyat

🖥️ Program Menü

Ana Menü:

1 - Kullanıcı Seç
2 - Yeni Kullanıcı
3 - Çıkış

Kullanıcı Menüsü:

1 - Hisseleri Listele
2 - Hisse Ekle
3 - Hisse Sil
4 - Hisse Transfer
5 - Net Değer
6 - Geri

📊 Net Değer Hesabı

Kullanıcının portföy değeri şu şekilde hesaplanır:

Toplam = Adet × Fiyat
